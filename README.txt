Разработан программный модуль на языке
программирования python для построения и применения линейного блокового
двоичного кода, способного обнаруживать и исправлять одиночные ошибки в
передаваемых данных:

По заданным с клавиатуры параметрам (n, k) программный
модуль строит линейный блоковый двоичный код, исправляющий
одну ошибку. Если для заданных параметров такого кода не существует, возвращается ошибка. Для найденного кода на экран выводится его
проверочная матрица, а также количество ошибок, которые код может
исправить. При построении кода проверяется границы Хэмминга,
Варшамова-Гилберта и Синглтона. Также модуль поддерживает
кодирование информации и её декодирование с помощью синдромов.
